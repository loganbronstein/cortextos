A clear setup for turning messy notes into a useful AI second brain.
At the centre is Andrej Karpathy’s LLM-wiki idea: keep raw source material, process what matters, and
slowly turn it into structured, connected knowledge. The support layer around it — structure, search, graph,
recurring jobs — is what makes it useful.
The stack at a glance
LAYER TOOL WHAT IT DOES
Vault Obsidian Local-first markdown notes. The home base.
Method Karpathy LLM wiki Capture raw → process → link → compound.
Structure PARA Projects, Areas, Resources, Archive.
Workflow Obsidian Skills Kepano’s patterns for working inside the vault.
Graph Graphify Surfaces clusters, key nodes, relationships.
Search QMD Better markdown retrieval as the vault grows.
